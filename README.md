# TombOfTheLichKing
Жанр: RPG-сапёр с мета-прогрессией и экономикой
1. Игровая механика:
  Основное поле это сетка N x N (начинается с 5 x 5 и далее по нарастающей)
  Есть разные типы клеток:
    Пустая: При открытии показывает цифры подсказки о количестве сокровищ и скелетов рядом
    Сундук: Даёт золото
    Скелет: Наносит урон
    Ключ: Необходимо его найти чтобы пройти на следующий уровень
  На каждом уровне игроку даётся ограниченное количество ходов, ходы тратятся при любом открытии клетки
2. RPG система:
  Есть две характеристики влияющие на геймплей:
    Телосложение: Определяет максимальное здоровье
    Удача: Даёт шанс активировать особый эффект при открытии клетки
  Также есть предметы (расходники), которые дают мгновенный результат
3. Экономика и прогрессия:
   1) Лагерь (Магазин): В нём можно начать новый уровень, тратить золото на прокачку характеристик и покупку предметов
   2) Уровень: На нём нужно искать ключ, собирать золото и стараться не умереть от скелетов
   3) Конец уровня: Получение базовой награды за прохождение плюс бонусное золото за оставшиеся ходы
